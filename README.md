#勤怠管理システム(Attendance-processing-my-first-repo-)

## プロジェクトの概要
主に社員登録、出勤/退勤打刻、勤務時間計算機能を提供する、シンプルなコマンドラインベースの勤怠管理システムです。
依存性の注入を導入しJavaのオブジェクト指向プログラミングの基本と、モダンな日付・時刻API（java.time）を利用して作成しました。

## 何ができるの？
- 機能1: 社員登録(社員IDと名前を登録し、システムに追加します)
- 機能2: 出勤打刻(社員IDを読み込み、LocalDateTimeを自動で取得して出勤時間を登録)
- 機能3: 退勤打刻(社員IDを読み込み、LocalDateTimeを自動で取得して退勤時間を登録)
- 機能4: 勤怠状況一覧表示(全社員の出退勤の状況の一括出力)
- 機能5: 実労働時間一覧表示(打刻された出勤、退勤時間に基づいて実労働時間を計算)

## 意識して取り組んだ点
- 努力1:クラスの細分化
- 努力2:依存性の注入 
- 努力3:どのような状況でも正確に処理できる設計(同一社員idでの社員登録防止、containsKey()による例外処理など)
- 努力4:現在の仕様に近づけるためにモダンなLocalDateTimeの使用
- 努力5:Mapによる社員idと社員情報、社員idと勤怠状況の紐づけ
  
## プログラムの改善性
- 改善案1:データの永久保存(現在のプログラムは終了してしまうと登録したデータが消えてしまう)
- 改善案2:同一人物による二重打刻防止
- 改善案3:休憩の選択肢を追加する

## 使い方
1. リポジトリをクローンする
2. 必要なライブラリをインストール
3. プログラムを実行
